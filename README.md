# iconit
Database of popular icons and tags.

## Development

### Requirements

- Node
- NPM
- Ruby
- Bundler

### Setup

- [sudo] npm install
- bundle install
- node_modules/bower/bin/bower install
- foreman start -f Procfile.dev

## Production

- foreman start

This automatically happens when deploying to heroku.
